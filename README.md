## Research

- **[LLM-as-a-Judge Criteria Ensembling](https://github.com/composo-ai/llm-judge-criteria-ensembling)** — An empirical investigation of practical LLM-as-a-Judge improvement techniques on RewardBench 2. Task-specific criteria injection combined with ensembling reaches up to 85.8% accuracy (+13.5pp over baseline), generalising across OpenAI and Anthropic models.
- **[DoG-Saliency](https://github.com/tobybreckon/DoG-saliency)** — A reference implementation of real-time salient object detection based on Division of Gaussians [Katramados/Breckon, 2011], authored during a research internship at Durham University.
